Simple DepthCloud Example

Run the following commands in the terminal then refresh this page.

  1. roscore
  2. roslaunch rosbridge_server rosbridge_websocket.launch
  3. roslaunch openni_launch openni.launch depth_registration:=true
  4. rosrun ros_web_video ros_web_video _port:=9999 _framerate:=15 _bitrate:=250000 _profile:=best www_file_server:=true _wwwroot:=/path/to/ros3djs/
  5. rosrun depthcloud_encoder depthcloud_encoder_node _depth:=/camera/depth_registered/image_rect _rgb:=/camera/rgb/image_rect_color
To see this page in your browser, go to http://localhost:9999/examples/depthcloud.html.
